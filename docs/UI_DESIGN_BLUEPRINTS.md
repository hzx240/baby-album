### 1.3 设计规范

**配色方案**：
- 拖拽区域：虚线边框 `border-2 border-dashed border-primary-300`
- 拖拽区域悬停：`bg-primary-50 border-primary-500`
- 进度条：渐变 `from-primary-500 to-accent-500`
- 成功图标：`text-green-500`
- 等待图标：`text-warm-500`

**间距**：
- 照片预览卡片间距：`gap-3` (12px)
- 卡片圆角：`rounded-xl` (12px)
- 章节间距：`my-6` (24px)

**动画**：
- 拖拽区域进入：`animate-bounce-soft`
- 进度条动画：`transition-all duration-300`
- 照片添加：`animate-scale-in`

## 6. 设计规范摘要

### 6.1 配色方案

```css
/* 主色调 - 玫瑰粉（温馨、爱意）*/
--primary-500: #f43f5e;

/* 辅助色 - 天空蓝（清新、希望）*/
--accent-500: #0ea5e9;

/* 点缀色 - 柠檬黄（活力、快乐）*/
--warm-400: #facc15;

/* 成功/错误/警告 */
--success: #22c55e;
--error: #ef4444;
--warning: #f59e0b;
```

### 6.2 圆角规范

```css
--radius-sm: 6px;    /* 小元素 */
--radius-md: 8px;    /* 按钮、输入框 */
--radius-lg: 12px;   /* 卡片 */
--radius-xl: 16px;   /* 大卡片 */
--radius-2xl: 24px;  /* 模态框 */
--radius-full: 9999px; /* 徽章、头像 */
```

### 6.3 阴影规范

```css
--shadow-sm: 0 1px 2px rgba(0,0,0,0.05);
--shadow-md: 0 4px 6px rgba(0,0,0,0.1);
--shadow-lg: 0 10px 15px rgba(0,0,0,0.1);
--shadow-glow: 0 0 20px rgba(244,63,94,0.15);
```

### 6.4 动画规范

```css
--duration-fast: 150ms;

--duration-slow: 500ms;

--ease-out: cubic-bezier(0,0,0.2,1);
```

## 7. 响应式设计

### 7.1 断点

```css
--breakpoint-sm: 640px;   /* 手机横屏 */
--breakpoint-md: 768px;   /* 平板 */
--breakpoint-lg: 1024px;  /* 小桌面 */
--breakpoint-xl: 1280px;  /* 桌面 */
```

### 7.2 照片网格列数

```css
/* 移动端: 2列 */
@media (max-width: 639px) {
  .photo-grid { grid-template-columns: repeat(2, 1fr); }
}

/* 平板: 3-4列 */
@media (min-width: 640px) and (max-width: 1023px) {
  .photo-grid { grid-template-columns: repeat(3, 1fr); }
}

/* 桌面: 5-6列 */
@media (min-width: 1024px) {
  .photo-grid { grid-template-columns: repeat(5, 1fr); }
}
```
