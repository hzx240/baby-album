## 🚨 新发现的 P0 安全风险

### 风险 1: COPPA 合规问题 (儿童在线隐私保护法)

| 属性 | 详情 |
|------|------|
| **问题** | Phase 2 已有 PhotoFace 模型（人脸特征向量） |
| **风险** | 🇺🇸 美国法律违规，高额罚款 |
| **罚款** | 每违规事件：$16,000 - $50,000+ |
| **影响** | **可能面临法律诉讼和巨额罚款** |

**必须修复** (16h):
- [ ] 添加明确同意机制（父母同意）
- [ ] AES-256 加密人脸特征向量
- [ ] 增加完全删除功能（用户请求）
- [ ] 更新隐私政策（COPPA 合规）

**优先级**: 🔴 **P0 - Week 1 必须完成**

---

### 风险 2: AI API 成本失控

| 属性 | 详情 |
|------|------|
| **问题** | API 密钥泄露 → 数千美元账单 |
| **成本** | 1000 用户：~$460/月 |
| **成本** | 5000 用户：~$2,050/月 |
| **风险** | 成本可能失控，超出预算 |

**必须修复** (12h):
- [ ] AWS Budget 自动暂停
- [ ] 用户配额控制
- [ ] API 密钥轮换（每月）
- [ ] 成本监控告警（每日/每周）

**优先级**: 🔴 **P0 - Week 1 必须完成**

---

### 风险 3: 视频上传安全漏洞

| 属性 | 详情 |
|------|------|
| **问题** | PRD 缺少文件验证，直接支持 500MB |
| **风险** | 恶意文件上传、服务器攻击 |
| **影响** | 系统安全漏洞 |

**必须修复** (20h):
- [ ] 五层验证：
  1. Magic Number (文件头)
  2. MIME 类型验证
  3. FFmpeg 视频验证
  4. 文件大小限制
  5. 病毒扫描
- [ ] 异步处理
- [ ] 用户隔离

**优先级**: 🔴 **P0 - Week 1 必须完成**

---

### 风险 4: 访问密码强度不足

| 属性 | 详情 |
|------|------|
| **问题** | 6 位数字太弱（GPU 1 秒破解） |
| **风险** | 暴力破解、未授权访问 |
| **影响** | 隐私泄露 |

**必须修复** (+4h):
- [ ] 从 6 位改为 8 位
- [ ] 允许字母 + 数字
- [ ] 防暴力破解（5 次锁定 30 分钟）

**优先级**: 🟡 **P1 - Week 1-2 完成**

## 💰 商业影响

### 成本增加

| 项目 | 原估算 | 新估算 | 增加 |
|------|--------|--------|------|
| Must Have | 100h | 168h | +68h |
| Should Have | 140h | 104h | -36h |
| Could Have | 100h | 28h | -72h |
| **总计** | **340h** | **300h** | **-40h** |

**结论**: 总工作量减少 40 小时，但 Must Have 增加 68h

### 收入机会 (AI 功能改为付费)

| 功能 | 价格 | 预期转化 |
|------|------|----------|
| AI 质量评分 | $5/月 | 10% 用户 |
| 智能场景分类 | $3/月 | 10% 用户 |
| 自动标签建议 | $2/月 | 10% 用户 |

**预期收入**: 假设 1000 用户，10% 付费
- 100 付费用户 × ($5+$3+$2) = **$1,000/月**

### 成本控制 (用户配额)

| 用户类型 | AI 配额/月 |
|---------|-----------|
| 免费用户 | 10 张 |
| 付费用户 | 100 张 |

**预期成本**:
- 免费用户：900 × 10 = 9,000 张
- 付费用户：100 × 100 = 10,000 张
- 总计：19,000 张/月
- 成本：19,000 × $0.001 = **~$19/月**

**vs 无控制**:
- 1000 用户 × 50 张/月 = 50,000 张
- 成本：50,000 × $0.001 = **~$50/月**

**节省**: $50 - $19 = **$31/月** (62%)
