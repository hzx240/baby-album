## [1.0.0] - 2026-02-14

### Added
- ✨ **核心功能**
  - 宝宝档案管理
  - 家庭成员管理
  - 照片上传与管理
  - 相册管理
  - 时间线浏览

- ✨ **批量上传系统**
  - 拖拽上传支持
  - 批量选择文件
  - 上传进度显示
  - 断点续传
  - 自动照片分类
  - EXIF 信息提取
  - 批量元数据编辑

- ✨ **智能相册**
  - 基于规则的自动相册
  - 5 种智能规则类型：
    - 时间范围规则
    - 人物标签规则
    - 重要日期规则
    - 地点规则
    - 自定义标签规则
  - 自动照片匹配
  - 相册统计信息

- ✨ **时间线增强**
  - 重要日期管理
  - 里程碑记录
  - 智能分组展示
  - 时间线统计
  - 照片时间轴

- ✨ **安全性**
  - JWT 认证
  - 家庭成员权限控制
  - 基于 Token 的邀请系统
  - 邀请过期控制
  - SHA-256 重复检测
  - 文件类型验证
  - 输入验证增强

- ✨ **DevOps**
  - Docker 容器化
  - Docker Compose 编排
  - GitHub Actions CI/CD
  - 自动化测试
  - 监控系统（Prometheus + Grafana）
  - 日志聚合（Loki）
  - 备份与恢复方案

- ✨ **性能优化**
  - Redis 缓存层
  - 图片三种规格（原图、压缩图、缩略图）
  - 数据库索引优化
  - 批量查询 API
  - 分页和字段选择

### Changed
- 🔄 **数据库**
  - 合并 Phase 1 和 Phase 2 Schema
  - 新增 11 张数据表
  - 优化索引和关系

- 🔄 **API**
  - 新增 18 个 Phase 2 端点

  - 改进错误处理

## 版本说明

### 版本格式

- **主版本号**: 不兼容的 API 变更
- **次版本号**: 向下兼容的功能新增
- **修订号**: 向下兼容的问题修复

### 变更类型

- **Added**: 新增功能
- **Changed**: 功能变更
- **Deprecated**: 即将废弃的功能
- **Removed**: 已删除的功能
- **Fixed**: Bug 修复
- **Security**: 安全问题修复
