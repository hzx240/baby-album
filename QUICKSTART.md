## 常见问题

### Q: 注册失败/无错误提示
A: 确保网络代理不会拦截 localhost 请求

### Q: 后端连接失败
A: 检查后端是否在端口 8888 运行

### Q: Redis 连接错误
A: 可以忽略，不是必需的（仅用于缓存优化）

## 开发说明

### 端口配置
- 后端默认: 8888 (可在 backend/.env 中修改 PORT)
- 前端默认: 5173 (Vite 自动选择可用端口)
- 数据库: SQLite (backend/prisma/dev.db)

### 环境变量
后端配置: `backend/.env`
前端配置: `frontend/.env` 或 `frontend/src/lib/constants.ts`

### 重启服务
```bash
# 停止服务: Ctrl+C

# 启动后端
cd backend
npm run start:dev

# 启动前端
cd frontend
npm run dev
```
